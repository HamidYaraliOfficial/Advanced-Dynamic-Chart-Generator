# Advanced Dynamic Chart Generator

---

## English

### Overview
**Advanced Dynamic Chart Generator** is a **professional-grade, fully interactive desktop application** built with **PyQt6** and **Matplotlib** that empowers users to **create stunning, publication-quality charts** from **dynamic data tables** — **in real time**.

With **10 chart types**, **4 languages**, **5 elegant themes**, **dynamic data editing**, **CSV import/export**, **advanced styling**, and **full RTL support**, this tool is perfect for **data analysts, scientists, educators, students, and developers** who need **beautiful, customizable visualizations** with **zero coding**.

---

### Key Features
- **10 Professional Chart Types**:
  - Bar • Line • Pie • Scatter • Area
  - Histogram • Box Plot • Heatmap • Radar • Polar
- **Dynamic Data Table**:
  - Add/remove rows & columns
  - Real-time editing
  - CSV import/export
- **5 Sample Datasets**:
  - Sales • Weather • Student Grades • Survey Results
- **Multilingual Interface**:
  - English • Persian (فارسی) • Chinese (中文) • Russian (Русский)
  - Full **RTL layout** support
- **5 Stunning Themes**:
  - Windows Default • Light • Dark • Red • Blue
- **Advanced Styling Controls**:
  - Font size • Line width • Marker size
  - Opacity • Label rotation • Color schemes
  - 3D view • Grid • Legend
- **Real-Time Chart Generation**
- **High-Quality Export** (PNG/JPEG)
- **Responsive & Scalable** UI
- **No external dependencies** beyond PyQt6 & Matplotlib

---

### Requirements
- Python 3.8+
- PyQt6
- Matplotlib
- Pandas
- NumPy

---

### Installation
1. Install dependencies:
   ```bash
   pip install PyQt6 matplotlib pandas numpy
   ```
2. Save the script as `chart_generator.py`
3. Run:
   ```bash
   python chart_generator.py
   ```

---

### Usage
1. Choose **language** and **theme**
2. Select **sample data** or create your own
3. Edit the **dynamic table**
4. Choose **chart type** and configure settings
5. Click **"Generate Chart"**
6. Use **"Save as Image"** or **"Export Data"**

> **Pro Tip**: Use **Heatmap** with pivot data or **Radar Chart** for multi-dimensional comparisons.

### 项目概览
**高级动态图表生成器** 是一款**专业级、完全交互式桌面应用程序**，使用 **PyQt6** 和 **Matplotlib** 构建，使用户能够从**动态数据表**中**实时创建惊艳、出版级图表**。

支持 **10 种图表类型**、**4 种语言**、**5 种优雅主题**、**动态数据编辑**、**CSV 导入/导出**、**高级样式控制**和**完整 RTL 支持**，此工具非常适合**数据分析师、科学家、教师、学生和开发者**，他们需要**美观、可定制的可视化**且**无需编码**。

---

### 核心功能
- **10 种专业图表类型**：
  - 柱状图 • 折线图 • 饼图 • 散点图 • 面积图
  - 直方图 • 箱线图 • 热力图 • 雷达图 • 极坐标图
- **动态数据表**：
  - 添加/删除行和列
  - 实时编辑
  - CSV 导入/导出
- **5 个示例数据集**：
  - 销售 • 天气 • 学生成绩 • 调查结果
- **多语言界面**：
  - 中文 • 英语 • 波斯语 • 俄语
  - 完整支持**从右到左 (RTL)**
- **5 种惊艳主题**：
  - Windows 默认 • 浅色 • 深色 • 红色 • 蓝色
- **高级样式控制**：
  - 字体大小 • 线宽 • 标记大小
  - 透明度 • 标签旋转 • 配色方案
  - 三维视图 • 网格 • 图例
- **实时图表生成**
- **高质量导出** (PNG/JPEG)
- **响应式与可缩放**界面
- **仅依赖 PyQt6 和 Matplotlib**
